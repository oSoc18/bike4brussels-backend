Brussels Mobility Routeplanner Open Data.

Every time a user plans a route on routeplanner.bike.brussels over the Brussels regional cycling network, the request is saved in a csv file of that date.

The files have 5 columns and contain the following data respectively: timestamp of request, latitude of starting position, longitude of starting position, latitude of ending position, longitude of ending position. The coordinates use the WSG84 coordinate system.

The files can be found here.
